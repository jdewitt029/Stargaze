Stargaze (2019)

Processing

This assortment of shapes and color was created using only lines and shaped in the coding software called Processing. This was an in-class assignment that I had to do and it was a fun exercise that helped me learn how to code. Prcoessing works like a graph with coordinates and that is how we place certain things in certain areas. This work was mostly just freestyling and this bizzare shape collection was the end result.
